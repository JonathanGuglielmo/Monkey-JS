JavaScript

Javascript es una especie de lenguaje de programación ligera, interpretado por la mayoría de los navegadores y que les
proporciona a las páginas web, efectos y funciones complementarias a las consideradas como estándar HTML Este tipo de lenguaje de
programación, con frecuencia son empleados en los sitios web, para realizar acciones en el lado del cliente, estando centrado en el
código fuente de la página web.
